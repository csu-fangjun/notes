Run GitHub actions locally¬∂

It is time-consuming to use GitHub action to test whether the current workflow is working. https://github.com/nektos/act provides a tool to test workflows locally. Refer to the above repository for installation.

Hint

The script https://github.com/nektos/act/blob/master/install.sh is worth reading.

Hello world¬∂

The following shows an exmaple workflow.

./code/actions/env.yml¬∂
name: My first workflow
on:
  push: [master]

jobs:
  foo_job:
    runs-on: ubuntu-latest
    steps:
      - name: Test 1
        env:
          FOO: hello
          BAR: world
        shell: bash
        run: |
          echo "foo is ${FOO}"
          echo "BAR is ${BAR}"

Every workflow has an optional name and some conditions on which it will run. A workflow consists of a list of jobs. A job is run on an environment specified by runs-on. Inside a job there are a list of steps.

The following shows how to use act to run the above workflow locally.

$ act -l -W ./actions/
Stage  Job ID   Job name  Workflow name      Workflow file  Events
0      foo_job  foo_job   My first workflow  env.yml        push
