ScalarType¶

See c10/core/ScalarType.h. and https://github.com/pytorch/pytorch/blob/master/torch/csrc/api/include/torch/types.h.

ScalarType is an enum class, i.e., enum class ScalarType : int8_t { ... }.

It has the following members:
