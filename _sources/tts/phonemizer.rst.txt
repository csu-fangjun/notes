phonemizer
==========

Its documentation is at `<https://bootphon.github.io/phonemizer/>`_.

See `<https://bootphon.github.io/phonemizer/install.html>`_ for installation.

On macOS, it is:

.. code-block:: bash

   brew install espeak
   pip install phonemizer

.. code-block:: bash

   DUMMY1/LJ049-0022.wav|The Secret Service believed

   # cleaned

.. code-block:: bash

  (py38) fangjuns-MacBook-Pro:~ fangjun$ echo "The Secret Service believed" | phonemize
