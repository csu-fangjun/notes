Node
====

Installation on macos
---------------------

`<https://nodejs.org/en/download>`_

.. code-block:: bash

   wget https://nodejs.org/dist/v18.16.0/node-v18.16.0-darwin-x64.tar.gz
   tar xvf node-v18.16.0-darwin-x64.tar.gz
   export PATH=/Users/fangjun/software/node-v18.16.0-darwin-x64/bin:$PATH
